# Contributing

## Setup

This repository uses:

- `node` via [`nvm`](https://github.com/nvm-sh)
- `pnpm` as package manager via [`corepack`].

```shell
# install nvm
nvm use
```

```shell
corepack use pnpm
```

## Development

To run dev server use:

```shell
pnpm dev
```

To run all checks locally:

```shell
pnpm codecheck
```
